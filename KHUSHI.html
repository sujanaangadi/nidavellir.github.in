2) AI-Based Worker Safety Monitoring

Algorithms Used: Dijkstra, BFS, Heap

Explanation:
  • Factory is modeled as a weighted graph where nodes = zones, edges = hazard levels.
  • Dijkstra finds the safest escape route with minimum accumulated risk.
  • BFS simulates hazard spreading such as smoke, gas, or fire expansion.
  • A priority queue highlights zones requiring urgent alarms.
Time Complexity: O(E log V)
Show Code 
#include 
using namespace std;

vector dijkstra(int n, int src, vector>> &adj){
    vector dist(n, INT_MAX);
    priority_queue, vector>, greater>> pq;

    dist[src] = 0;
    pq.push({0,src});

    while(!pq.empty()){
        auto [d,u] = pq.top(); pq.pop();
        if(d > dist[u]) continue;

        for(auto &e : adj[u]){
            int v=e.first, w=e.second;
            if(dist[u] + w < dist[v]){
                dist[v] = dist[u] + w;
                pq.push({dist[v], v});
            }
        }
    }
    return dist;
}

3) Predictive Maintenance for Industrial Machines

Algorithms Used: Segment Tree, Heap, DFS

Explanation:
  • Segment trees store machine vibration logs and allow fast maximum queries.
  • A sudden spike in vibration signals potential mechanical failure.
  • DFS maps dependency among machine components (which part affects which).
  • Heap ranks machines by urgency for scheduled maintenance.
Time Complexity: O(log n) per query

using namespace std;

int build(vector& a, vector& st, int i, int l, int r){
    if(l==r) return st[i]=a[l];
    int m=(l+r)/2;
    return st[i]=max(build(a,st,2*i+1,l,m),
                     build(a,st,2*i+2,m+1,r));
}
