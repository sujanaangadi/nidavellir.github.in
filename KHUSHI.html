1) Smart Textile Quality Detector

Algorithm Used: Threshold-Based Defect Detection

Show Algorithm 
for each sensor_value:
    if |value - expected| > threshold:
         mark defect
C++ Code: 
#include 
#include 
using namespace std;

int main() {
    int expected = 100, threshold = 20;
    int fabric[6] = {100, 102, 98, 150, 101, 99};

    for(int i=0; i<6; i++){
        if(abs(fabric[i] - expected) > threshold)
            cout << "Defect at position " << i << endl;
    }
}

5) Smart Inventory Forecasting

Algorithm Used: Moving Average Forecasting

Show Algorithm 
forecast = (sum of previous N months) / N
C++ Code: 
#include 
using namespace std;

int main(){
    int data[6]={50,60,55,65,70,80};
    int sum=0;

    for(int i=0;i<6;i++)
        sum += data[i];
