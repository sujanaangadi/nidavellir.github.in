1) Smart Traffic Signal System

Algorithms Used: Max-heap Algorithm

    How the Algorithm Works

    1. Traffic cameras/sensors collect live vehicle count per lane.
    2. Emergency sensor detects if a special vehicle (like ambulance, fire trucks, police van etc.) is present.
    3. All lanes are inserted into a priority queue (max-heap) where:
      • Emergency lane gets highest key value
      • Then highest vehicle count
      • Then safety timeout value
    4. The top element of heap is extracted → that lane gets green signal.
    5. Signal duration = number of vehicles × dynamic factor.
    6. After processing the lane:
      • Vehicle count is decreased
      • Priority queue is heapified again
    7. Cycle continues until all lanes are cleared fairly.
    Efficiency Analysis:
    Time Complexity: O(n)
    Space Complexity: O(n)
    SDGs mapped: 3, 9, 11, 13 Show Code 
    #include 
#include 
#include 
using namespace std;

struct Lane {
    int lane_no;
    int vehicle_count;
    bool emergency;
};

struct Compare {
    bool operator()(Lane const &a, Lane const &b) {
        if (a.emergency != b.emergency)
            return b.emergency > a.emergency;
        return a.vehicle_count < b.vehicle_count;
    }
};

int main() {
    while (true) {
        int n;
        cout << "\nEnter number of lanes (0 to exit): ";
        cin >> n;
        if (n == 0) break;

        priority_queue, Compare> pq;

        for (int i = 1; i <= n; i++) {
            int count;
            char emer;
            cout << "Enter vehicle count for Lane " << i << ": ";
            cin >> count;
            cout << "Emergency vehicle? (y/n): ";
            cin >> emer;
            pq.push({i, count, (emer == 'y' || emer == 'Y')});
        }

        Lane top = pq.top();
        int signal_time = top.vehicle_count * 2;
        if (signal_time < 20) signal_time = 20;

        cout << "\nLane chosen: " << top.lane_no << endl;
        cout << "Vehicle count: " << top.vehicle_count << endl;
        if (top.emergency) cout << "Emergency vehicle detected → PRIORITY GIVEN!\n";
        cout << "Green duration: " << signal_time << " seconds\n";
    }

    cout << "\nSystem Stopped.\n";
    return 0;
}

2) Smart Waste Management System

Algorithms Used: Pattern Classification Algorithm
    How the Algorithm Works:
    1. The system takes the detected waste type as input from a sensor or camera.
    2. Pattern Matching / Classification
      The input is compared against predefined patterns associated with each waste category (Plastic bin, Organic bin, Paper bin, Metal bin).
    3. Bin Mapping
      Each category is mapped to a specific bin using a lookup table.
    4. Decision Output
      The system prints the detected waste type, category, and assigned bin.
      Unknown items are sent to a manual inspection bin.
    Efficiency Analysis:
    Time Complexity: O(1) — direct mapping using decision logic
    Space Complexity: O(1) — fixed dictionary of categories
    SDGs mapped: 3, 11, 12, 13 Show Code 
    #include 
#include 
using namespace std;

int classifyWaste(const string &input) {
    // Pattern classes — keywords mapped to category ID
    if (input == "plastic bottle" || input == "polythene" || input == "plastic")
        return 1;
    if (input == "banana peel" || input == "food leftovers" || input == "organic")
        return 2;
    if (input == "newspaper" || input == "cardboard" || input == "paper")
        return 3;
    if (input == "aluminium can" || input == "iron piece" || input == "metal")
        return 4;
    return 0; // unknown
}

int main() {
    unordered_map bins = {
        {1, "Plastic Bin"},
        {2, "Organic Bin"},
        {3, "Paper Bin"},
        {4, "Metal Bin"}
    };

    string waste;
    cout << "Enter the detected waste type: ";
    getline(cin, waste);

    int category = classifyWaste(waste);

    if (category == 0) {
        cout << "Unknown material → Send to Manual Inspection Bin";
    } else {
        cout << "Detected: " << waste << endl;
        cout << "Classification Category: " << category << endl;
        cout << "Dropping into ➤ " << bins[category];
    }
    return 0;
}

3) Public Transport Tracking Simulator

Algorithms Used:Dijkstra's Algorithm
    • Calculate Fastest Path
    • Represent all stops as nodes
    • Roads between stops as edges
    • Distance/time/traffic as weights

    Dijkstra:

    1. Start from current stop (example: Stop A)
    2. Check all neighbors
    3. Pick the one with minimum travel cost
    4. Repeat until Stop B is reached
    • Prevents buses from taking long or blocked roads
    • Minimizes delays
    • Reduces fuel usage
    • Helps in real-time dynamic routing
    Efficiency: O(n) SDG: 9, 11, 13, 3 Show Code

4) Digital Food Ordering & Billing

Algorithms Used:Hash Table
  • Food ID becomes the key in the hash table
  • A hash function converts the key into a bucket number
  • The item is stored in that bucket
  • When ordering, menu[id] jumps directly to the bucket
  • The item is found in O(1) time (very fast)
  • Billing uses this fast lookup to get price and calculate totals
Efficiency: O(1) SDG: 9,12, 8,11 Show Code

using namespace std;

struct Category {
    int id;
    string name;
    double price;
};

int main() {
    unordered_map menu = {
        {1, {1, "Royal Indian Platter", 950.0}},
        {2, {2, "Continental Classics", 1100.0}},
        {3, {3, "Chef’s Signature Degustation Menu", 1600.0}},
        {4, {4, "Exotic Beverages & Mocktails", 450.0}},
        {5, {5, "Gourmet Desserts & Patisserie", 520.0}}
    };

    cout << "========== MENU ==========\n";
    for (auto &m : menu)
        cout << m.second.id << ". " << m.second.name 
             << " — ₹ " << m.second.price << "\n";

    vector> order = {
        {2, 1},
        {4, 2}
    };

    cout << "\n----------- ORDER SUMMARY -----------\n";
    double subtotal = 0;

    for (auto &o : order) {
        Category item = menu[o.first];
        double cost = item.price * o.second;
        cout << item.name << " × " << o.second << " = ₹ " << cost << "\n";
        subtotal += cost;
    }

    double gst = subtotal * 0.05;
    double service_charge = subtotal * 0.10;
    double total = subtotal + gst + service_charge;

    cout << "\nSubtotal: ₹ " << subtotal;
    cout << "\nGST (5%): ₹ " << gst;
    cout << "\nService Charge (10%): ₹ " << service_charge;
    cout << "\n--------------------------------------";
    cout << "\nTOTAL BILL: ₹ " << total << "\n";

    return 0;
}
